BB84 Quantum Key Distribution Simulator

A Python-based simulation of the BB84 protocol, a quantum cryptographic method for secure key exchange. This simulator allows users to:

  • Generate secure keys using quantum principles
  • Transmit and encrypt secret messages
  • Simulate eavesdropping attempts by an attacker (Eve)
  • Analyze the effectiveness of eavesdropping detection
